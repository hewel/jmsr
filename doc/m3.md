**Reference: JMSR Design System (Material Design 3)**

We have established a centralized CSS system based on Panda CSS and Material Design 3 (Dark Theme).
**Do not** manually build components using raw utility classes if a recipe exists. Use the predefined recipes and text styles from `panda.config.ts`.

---

## Getting Started

Import the `css` function and recipes from the styled-system:

```tsx
import { css, cx } from '../../styled-system/css';
import { button, card, input } from '../../styled-system/recipes';
```

## Typography (Text Styles)

Use the `textStyle` property to apply M3 type scale.

### Display (Large Headers)
```tsx
css({ textStyle: 'displayLarge' })   // 57px/64px - Hero sections
css({ textStyle: 'displayMedium' })  // 45px/52px - Major headings
css({ textStyle: 'displaySmall' })   // 36px/44px - Section headings
```

### Headline (Section Headers)
```tsx
css({ textStyle: 'headlineLarge' })  // 32px/40px - Page titles
css({ textStyle: 'headlineMedium' }) // 28px/36px - Section titles
css({ textStyle: 'headlineSmall' })  // 24px/32px - Subsection titles
```

### Title (Card/Dialog Titles)
```tsx
css({ textStyle: 'titleLarge' })   // 22px/28px - Dialog titles
css({ textStyle: 'titleMedium' })  // 16px/24px - Card titles, font-medium
css({ textStyle: 'titleSmall' })   // 14px/20px - List item titles, font-medium
```

### Body (Reading Text)
```tsx
css({ textStyle: 'bodyLarge' })   // 16px/24px - Default text
css({ textStyle: 'bodyMedium' })  // 14px/20px - Secondary text
css({ textStyle: 'bodySmall' })   // 12px/16px - Captions
```

### Label (Buttons, Captions)
```tsx
css({ textStyle: 'labelLarge' })  // 14px/20px - Button text, font-medium
css({ textStyle: 'labelMedium' }) // 12px/16px - Tabs, badges, font-medium
css({ textStyle: 'labelSmall' })  // 11px/16px - Timestamps, font-medium
```

## Buttons (Recipe)

Use the `button` recipe with variants:

### Primary Action (High Emphasis)
```tsx
<button class={button({ variant: 'primary' })}>Connect</button>
```
- **Usage:** The single most important action on a screen (e.g., "Connect", "Save", "Play").
- **Style:** Filled, `backgroundColor: 'primary'`, `color: 'onPrimary'`, rounded-full, height 40px.

### Tonal Action (Medium Emphasis)
```tsx
<button class={button({ variant: 'tonal' })}>Cancel</button>
// or
<button class={button({ variant: 'secondary' })}>Cancel</button>
```
- **Usage:** Alternative actions, secondary buttons.
- **Style:** `backgroundColor: 'secondaryContainer'`, `color: 'onSecondaryContainer'`, rounded-full.

### Outlined Action (Low Emphasis)
```tsx
<button class={button({ variant: 'outlined' })}>Learn More</button>
```
- **Usage:** Less prominent actions that need visibility.
- **Style:** Transparent with `borderColor: 'outline'`, `color: 'primary'`, rounded-full.

### Text Action (Lowest Emphasis)
```tsx
<button class={button({ variant: 'text' })}>Forgot Password?</button>
```
- **Usage:** Inline actions, links within forms.
- **Style:** No background, `color: 'primary'`, minimal padding.

### Icon Button
```tsx
<button class={button({ variant: 'icon' })}>
  <RefreshCw class={css({ width: '20px', height: '20px' })} />
</button>
```
- **Usage:** Toolbar buttons, playback controls.
- **Style:** 40x40px, rounded-full, subtle hover state.

## Inputs (Recipe)

Use the `input` recipe with variants:

### Filled Input (M3 Standard)
```tsx
<input type="text" class={input({ variant: 'filled' })} placeholder="Enter value..." />
```
- **Style:** `backgroundColor: 'surfaceContainerHighest'`, bottom border only, rounded-t-lg.
- **Height:** 56px.

### Outlined Input
```tsx
<input type="text" class={input({ variant: 'outlined' })} placeholder="Enter value..." />
```
- **Style:** Transparent with `borderColor: 'outline'`, rounded-lg.
- **Height:** 56px.

### Default Input
```tsx
<input type="text" class={input({ variant: 'default' })} placeholder="Enter value..." />
```
- **Style:** `backgroundColor: 'surfaceContainer'`, full border, rounded-lg.
- **Height:** 48px.

### Input with Label Pattern
```tsx
<div class={css({ display: 'flex', flexDirection: 'column', gap: '4px' })}>
  <label class={css({ textStyle: 'labelMedium', color: 'onSurfaceVariant' })}>Server URL</label>
  <input type="text" class={input({ variant: 'filled' })} placeholder="http://..." />
</div>
```

## Cards (Recipe)

Use the `card` recipe with variants:

### Elevated Card
```tsx
<div class={card({ variant: 'elevated' })}>
  <h3 class={css({ textStyle: 'titleMedium' })}>Card Title</h3>
  <p class={css({ textStyle: 'bodyMedium', color: 'onSurfaceVariant' })}>Description</p>
</div>
```
- **Usage:** Floating elements, modals, important content.
- **Style:** `backgroundColor: 'surfaceContainerLow'`, shadow-md, hover:shadow-lg.

### Filled Card (Default)
```tsx
<div class={card({ variant: 'filled' })}>...</div>
```
- **Usage:** Setting groups, media items, standard containers.
- **Style:** `backgroundColor: 'surfaceContainerHighest'`, no shadow.

### Outlined Card
```tsx
<div class={card({ variant: 'outlined' })}>...</div>
```
- **Usage:** When clear visual separation is needed.
- **Style:** `backgroundColor: 'surface'`, `borderColor: 'outlineVariant'`.

## Icons

Use **Lucide Solid** icons (from `lucide-solid` package).

```tsx
import { Play, Settings, RefreshCw } from 'lucide-solid';

// Standard size
<Play class={css({ width: '20px', height: '20px' })} />
<Settings class={css({ width: '24px', height: '24px' })} />

// With color
<Play class={css({ width: '20px', height: '20px', color: 'primary' })} />
<Settings class={css({ width: '24px', height: '24px', color: 'onSurfaceVariant' })} />
```

- **Standard sizes:** 20px (inline), 24px (standalone).
- **Color:** Inherits from parent by default, or use `color` property.

## State Layers

M3 uses opacity-based state layers for interactive feedback. Use color modifiers:

| State | Opacity | Example |
|-------|---------|---------|
| Hover | 8% | `backgroundColor: 'primary/8'` |
| Focus | 12% | `backgroundColor: 'primary/12'` |
| Pressed | 12% | `backgroundColor: 'primary/12'` |
| Disabled | 38% opacity on content | `color: 'onSurface/38'` |

Custom scrollbars are automatically applied via globalCss in panda.config.ts:
- Track: `backgroundColor: 'surfaceContainer'`
- Thumb: `backgroundColor: 'outlineVariant'`, rounded
- Thumb hover: `backgroundColor: 'outline'`

## Combining Styles with cx

Use `cx` to combine multiple class strings:

```tsx
import { css, cx } from '../../styled-system/css';
import { button } from '../../styled-system/recipes';

<button class={cx(
  button({ variant: 'primary' }),
  css({ width: '100%', marginTop: '16px' })
)}>
  Submit
</button>
```

## Quick Reference

### Page Structure
```tsx
<div class={css({ minHeight: '100vh', backgroundColor: 'background', padding: '24px' })}>
  <div class={css({ maxWidth: '672px', marginX: 'auto' })}>
    <h1 class={css({ textStyle: 'headlineLarge', color: 'onSurface', marginBottom: '16px' })}>
      Page Title
    </h1>
    <div class={cx(card({ variant: 'filled' }), css({ marginBottom: '24px' }))}>
      <h2 class={css({ textStyle: 'titleMedium', marginBottom: '16px' })}>Section</h2>
      {/* content */}
    </div>
  </div>
</div>
```

### Form Pattern
```tsx
<form class={css({ display: 'flex', flexDirection: 'column', gap: '16px' })}>
  <div class={css({ display: 'flex', flexDirection: 'column', gap: '4px' })}>
    <label class={css({ textStyle: 'labelMedium', color: 'onSurfaceVariant' })}>Label</label>
    <input type="text" class={input({ variant: 'filled' })} placeholder="..." />
  </div>
  <button type="submit" class={cx(button({ variant: 'primary' }), css({ width: '100%' }))}>
    Submit
  </button>
</form>
```

### Status Indicators
```tsx
// Success
<span class={css({ color: 'tertiary' })}>Connected</span>

// Error
<span class={css({ color: 'error' })}>Disconnected</span>

// Neutral
<span class={css({ color: 'onSurfaceVariant' })}>Pending</span>
```
